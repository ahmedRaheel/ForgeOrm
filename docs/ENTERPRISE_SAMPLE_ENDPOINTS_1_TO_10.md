# Enterprise sample endpoints 1-10

The sample API now exposes dedicated endpoints under `/enterprise/*` for:

1. Complex query power
2. Query performance engine
3. Advanced mapping
4. Enterprise DataFrame features
5. Migration/schema management
6. Multi-tenant support
7. Security
8. Bulk sync
9. Event/outbox support
10. Developer experience APIs

Examples include EXISTS/NOT EXISTS, subqueries, JSON/full-text predicates, lateral/CROSS APPLY, temporal SQL, profiling, compiled query, cache, streaming, Analyze/Explain/Validate, DTO/dictionary/JSON mapping, DataFrame cleaning/join/export, schema script generation, tenant-aware query/outbox, SQL safety validation, bulk sync strategies, outbox save, ToSql/ToDebugSql/Clone/Tag/Comment/Timeout.
